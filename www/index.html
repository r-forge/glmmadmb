glmmADMB

Please visit the following webpages for information about the glmmADMB package:

  • R-Forge (current development)
  • ADMB Project (recent homepage)
  • Otter Research (old homepage)

To install, run: 

install.packages("glmmADMB", repos="http://r-forge.r-project.org")

If you catch the R-Forge package server on a bad day, so the normal installation doesn't work, try: 

install.packages("glmmADMB", repos="http://r-forge.r-project.org", type="source")

———

Note that glmmADMB, like all R-Forge packages, requires the latest version of R (especially in Windows).
