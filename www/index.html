glmmADMB

Please visit the following webpages for information about the glmmADMB package:

  • R-Forge (current development)
  • ADMB Project (recent homepage)
  • Otter Research (old homepage)

To install, run: 

install.packages("glmmADMB", repos="http://r-forge.r-project.org")

If you catch the R-Forge package server on a bad day, so the normal installation doesn't work, try: 

install.packages("glmmADMB", repos="http://r-forge.r-project.org", type="source")

You can also install an alpha release of a new version of glmmADMB as follows: 

install.packages("glmmADMB",repos="http://glmmadmb.r-forge.r-project.org/repos",type="source")
The new version of glmmADMB has the following major changes:
  • new formula format, similar to that of the lme4 package, where random and fixed effects are specified as part of a single formula
  • multiple grouping variables (random effects) are allowed
  • wider range of distributions and link functions supported (e.g. binomial with N> 1)
The alpha release gives practically identical answers to the previous (stable) releases; however, for the time being it is somewhat slower than the stable release.

———

Note that glmmADMB, like all R-Forge packages, requires the latest version of R (especially in Windows).
